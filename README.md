A React single-page application for the LegalLink platform, built with Vite and React Router.

### Prerequisites

- **Node.js**: v18 or later is recommended  
- **npm**: comes with Node.js

You can check your versions with:

```bash
node -v
npm -v
```

### Install dependencies

From the project root (where `package.json` is):

```bash
npm install
```

### Run the app in development

```bash
npm run dev
```

Then open the URL shown in the terminal (by default `http://localhost:5173`).

### Build for production

```bash
npm run build
```

This outputs static files to the `dist` folder.

### Preview the production build

After building:

```bash
npm run preview
```

This serves the built app locally so you can test the production build.
